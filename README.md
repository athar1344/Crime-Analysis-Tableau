# Crime Analysis Using Tableau

## Project Overview
This project analyzes historical crime data using Tableau to identify patterns, trends, and insights that support law enforcement decision-making.

## Objectives
- Analyze overall crime distribution
- Study time-based crime patterns
- Identify long-term crime trends
- Perform comparative analysis (arrest, domestic, severity)
- Present insights using dashboards and storytelling

## Tools Used
- Tableau Desktop
- Tableau Public

## Dashboards
1. Overall Crime Statistics
2. Time Period Analysis
3. Trend Analysis
4. Comparative Analysis
